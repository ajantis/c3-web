c3-web
======

Web client for C3 system: https://github.com/ifunsoftware/c3-next

Prerequisites:
* Maven 3 installed 
* JDK 1.7 installed

To start a C3-Web locally just type in console:

```
cd c3-web
mvn clean jetty:run
```
 
This will start web application in embedded Jetty server listening on port 8080
